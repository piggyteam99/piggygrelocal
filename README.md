# Piggy GRE Manager

## نصب و اجرا (Installation & Run)

برای نصب و راه‌اندازی اسکریپت، دستورات زیر را در ترمینال وارد کنید:

```bash
curl -fsSL https://raw.githubusercontent.com/piggyteam99/piggygrelocal/main/piggygre.sh -o piggygre.sh && chmod +x piggygre.sh && ./piggygre.sh
```

---

## معرفی اسکریپت

این ابزار جهت ساخت لینک **IP** لوکال بین دو سرور طراحی شده است. با ایجاد یک تونل **GRE** بین دو سرور (ایران و خارج)، به هر دو طرف **IP** لوکال خصوصی اختصاص می‌یابد تا یک مسیر ارتباطی کاملاً خصوصی و پایدار برقرار شود.

## ویژگی‌ها و خروجی‌ها

* **اتصال پایدار:** ایجاد اینترفیس **gre1** و اختصاص **IP** های لوکال (**30/**).
* **اجرای خودکار:** دارای سرویس **systemd** جهت بالا آمدن خودکار تونل پس از هر بار ریبوت سرور.
* **مدیریت آسان:** امکان حذف کامل تنظیمات از طریق گزینه **Full Remove** در داخل منو.

## راهنمای استفاده

* **دسترسی سریع:** پس از اولین نصب، هر زمان که بخواهید می‌توانید با تایپ دستور **piggygre** منو را بالا بیاورید.
* **کاربرد:** پس از ساخته شدن **IP** های لوکال، می‌توانید انواع تانل‌ها از جمله **SSH**، **Rathole**، **GOST**، **WireGuard**، **OpenVPN** و **V2Ray** را روی این بستر اجرا کنید.
